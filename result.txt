=== TRACE.TXT ERROR RESOLUTION ===

Root Cause: Starlette 1.3.1 changed Jinja2Templates.TemplateResponse signature
  OLD: TemplateResponse(name: str, context: dict)
  NEW: TemplateResponse(request: Request, name: str, context: dict | None = None)

The old code called templates.TemplateResponse("landing.html", {"request": request})
which passed a dict as the `name` parameter, causing:
  TypeError: unhashable type: 'dict'

Fix applied: All 20 TemplateResponse calls updated to:
  templates.TemplateResponse(request, "landing.html")
The request param is now explicit; context["request"] is auto-set by Starlette.
